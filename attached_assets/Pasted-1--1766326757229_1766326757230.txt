1️⃣ فكرة الموقع

الموقع عبارة عن منصة إلكترونية لمتدربي التكوين المهني، تمكنهم من:

تقديم مشاكلهم بشكل منظم وسريع

تتبع الشكايات التي أرسلوها برقم الشكاية

عرض جميع المشاكل المدخلة في لوحة عرض مبسطة

التواصل مع إدارة الموقع من خلال قسم الاتصال

كل ذلك في صفحة واحدة (Single Page Application) مع ناف بار للتنقل بين الأقسام.

2️⃣ أقسام الموقع
أ. الرئيسية (Home)

تعرض رسالة ترحيبية وتعريف بالمنصة

توجيه المستخدم لاختيار القسم المناسب من القائمة

ب. تقديم مشكل (Report)

نموذج لإدخال بيانات الشكاية:

اسم المعهد

الشعبة / التخصص

نوع المشكلة (إداري، مكون/أستاذ، تجهيزات، توقيت، سلوك، أخرى)

وصف المشكل

زر لإرسال الشكاية

عند الإرسال، تظهر رسالة نجاح ✅

البيانات تُخزن مؤقتًا في LocalStorage (داخل المتصفح)

ج. تتبع الشكاية (Track)

المستخدم يدخل رقم الشكاية (الرقم يظهر تلقائي عند إرسال الشكاية)

الموقع يعرض تفاصيل الشكاية إذا كان الرقم صحيح

إذا الرقم غير موجود، تظهر رسالة خطأ باللون الأحمر

د. لوحة الشكايات (Dashboard)

تعرض جميع الشكايات المخزنة في LocalStorage

كل شكاية تحتوي على:

رقم الشكاية

نوع المشكلة

اسم المعهد

الشعبة

الوصف

تاريخ الإرسال

هـ. صفحة الاتصال (Contact)

تعرض معلومات للتواصل مع الإدارة: بريد إلكتروني، هاتف

3️⃣ تصميم الموقع

تصميم رايف (Neumorphism / Soft UI):

ظلال ناعمة تجعل العناصر تبدو بارزة أو منغرسة

ألوان هادئة ومريحة للعين (#e0e5ec ولون نص #333 أو أزرق)

تأثيرات أزرار وعناصر عند التركيز Hover/Active

ناف بار علوي للتنقل بين الأقسام، يبرز القسم النشط

4️⃣ البرمجة والتقنيات المستخدمة

HTML: لإنشاء الهيكل العام والأقسام

CSS: لتصميم رايف، تنسيق النصوص، الأزرار، الظلال، الخلفيات

JavaScript:

التنقل بين الأقسام بدون إعادة تحميل الصفحة

تخزين الشكايات في LocalStorage

عرض الشكايات وتتبعها بالرقم

عرض رسائل النجاح والخطأ

ملاحظة: الموقع حاليًا غير متصل بقاعدة بيانات خارجية، لذا عند مسح المتصفح تُفقد الشكايات. يمكن لاحقًا ربطه بـ PHP أو Python + قاعدة بيانات لجعل التخزين دائمًا.

5️⃣ ميزات الموقع

Single Page Application: الانتقال بين الأقسام سريع بدون إعادة تحميل الصفحة

Soft UI / Neumorphism: تصميم عصري وجذاب

تخزين محلي مؤقت: تجربة حقيقية لإرسال وتتبع الشكايات

سهل الاستخدام: لكل المتدربين بدون الحاجة لتسجيل دخول

متعدد الأقسام: تقديم مشكل، تتبع، عرض الشكايات، الاتصال
